1. Overview
Build an end-to-end automated pipeline that takes a single topic as input and produces a fully scripted, voiced audio reel in Dr. Kiran&apos;s style — his personality, language, energy, and voice — ready for video production.

Today, creating one reel requires manually writing a script, recording audio, and editing. This system eliminates all of that. One topic in. Kiran-style audio out.

Creating one reel takes hours of manual work — scripting, recording, editing

6. Personality Engine — Kiran&apos;s Profile
The personality engine is the core of the system. It encodes everything that makes Kiran sound like Kiran. Built from 9+ analyzed reference videos and his production spec.

Script Structure (every reel follows this)

Hook Types
Type A — Voiced parent question: everyday concerns (most common)
Type B — Shock statement or image: serious / safety topics
Type C — Performance interrupt: rare, exaggerated then snaps serious

Voice &amp; Tone Rules
Default tone: calm, warm, authoritative — never shouts or over-emotes
Language: natural Hindi/Hinglish — technical terms in English, delivery in Hindi
Filler words: &apos;toh&apos;, &apos;matlab&apos;, &apos;basically&apos;, &apos;actually&apos;, &apos;dekho&apos;
Addresses parents directly: &apos;parents&apos;, &apos;aap&apos;, &apos;aapka baby&apos;
Sentences are short and punchy
Emotion comes from editing + writing — not vocal performance

9. Phase 1 — Script Generator (Detailed)

User Flow
User opens web app
Types a topic (Hindi or English)
Selects hook type (Auto / A / B / C) and severity (Auto / Everyday / Serious)
Clicks Generate
Sees full script broken into sections with emotion tags, b-roll cues, caption highlights
Can copy the full script with one click

Script Output Format
Section label + time range (e.g. THE HOOK · 0:00–0:03)
Emotion tag per section (ENERGETIC / CAUTIONARY / CALM / WARM)
Script text with yellow highlights on key words
B-roll cue per section (what to show visually)
Stats: word count, estimated duration, hook type used
CTA recommendation badge for serious topics

Technical Stack
Frontend: React (single page app)
API: Claude Sonnet 4 via Anthropic API
System prompt: Kiran&apos;s full personality profile encoded
Output: JSON parsed and rendered as structured UI

10. Phase 2 — Voice Generation (Detailed)

Voice Cloning Setup
Extract clean audio from 20-30 Kiran videos (remove background music)
Upload to ElevenLabs Professional Voice Clone
Minimum 10-15 minutes of clean audio needed
Multiple emotional tones captured: calm, energetic, serious

Audio Generation Flow
Script sections fed to ElevenLabs with emotion parameters
Each section rendered with matching voice tone
Sections stitched into one continuous audio file
Output: downloadable .mp3, 55-60 seconds

Technical Stack
Voice cloning: ElevenLabs Professional Voice Clone API
TTS: ElevenLabs Text-to-Speech API
Audio processing: ffmpeg for stitching sections
Storage: Google Drive or local download
